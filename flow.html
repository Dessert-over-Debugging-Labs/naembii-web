🍳 cook — 화면 플로우 보드

유튜브 레시피를 핸즈프리로 따라 하는 요리 어시스턴트 · 전체 화면과 화면 간 이동 흐름 정리

화면 (Screen) 오버레이 / 모달 (Overlay) → 초록 = 이동할 화면
1
홈
Screen
  • ＋ 버튼→② 레시피 등록
  • 레시피 카드→④ 레시피 상세
2
레시피 등록
Overlay
  • 입력 완료→③ 분석 로딩
  • 취소 · 배경→① 홈
3
분석 로딩
Screen
  • 분석 완료 (자동)→④ 레시피 상세
4
레시피 상세
Screen
  • 요리 시작하기→⑤ 기본 조리 모드
  • ‹ 뒤로→① 홈
5
기본 조리 모드
Screen
  • 스크롤 · 스와이프 · 카드→단계 전환(같은 화면)
  • 타이머→⑦ 타이머 설정
  • 재료 보기→⑥ 재료 시트
  • 상단 ? →⑧ 음성 명령 가이드
  • 마지막 단계 다음→⑩ 완료
6
재료 시트
Overlay
  • ✕ · 배경→⑤ 기본 조리 모드
7
타이머 설정
Overlay
  • 시작→⑤ 조리(타이머 작동)
  • ✕ · 배경→⑤ 기본 조리 모드
8
음성 명령 가이드
Overlay
  • 알겠어요 · ✕→⑤ 기본 조리 모드
9
핸즈프리 모드
Screen
  • 음성 대화→AI 응답 · 단계 진행
  • 핸즈프리 종료→⑤ 기본 조리 모드
10
완료
Screen
  • 홈으로→① 홈
  • 다시 요리하기→⑤ 기본 조리 모드
11
후기 전체
Screen
  • 상세 · 후기 전체보기→진입
  • 후기 쓰기→후기 작성 시트(오버레이)
  • ‹ 뒤로→④ 레시피 상세
