# MVP 버튼 클릭 검증 매트릭스

작성일: 2026-07-01

## 범위

이번 라운드는 최신 변경과 직접 연결된 핵심 버튼을 CDP로 실제 클릭 검증했다. 전체 버튼 완전 매트릭스는 `MVP-06` Ralph 스토리로 남긴다.

## 검증 결과

| 화면 | 버튼 | 기대 액션 | 결과 |
| --- | --- | --- | --- |
| 홈 | 영상 링크로 바로 요리하기 | 레시피 등록 시트 열림 | 통과 |
| 홈 | 검색 | 검색 생략 안내 토스트 | 통과 |
| 홈 하단탭 | 등록 | 레시피 등록 시트 열림 | 통과 |
| 상세 | 요리 시작하기 | cook3 조리 모드 진입 | 통과 |
| 조리 모드 | 요리 완성 | 완료 화면 진입 | 통과 |
| 완료 | 다른 사용자를 위한 팁 남기기 | 팁 남기기 화면 진입 | 통과 |
| 팁 남기기 | 팁 등록 | 완료 화면 복귀 + 등록 토스트 | 통과 |
| 완료 | 후기 남기기 | 후기 작성 시트 열림 | 통과 |

## 남은 검증

- 홈 카테고리 7개, 섹션 전체 버튼 4개, 상세 탭 3개, 리뷰 작성 별점/사진, 타이머 프리셋, 재료 시트, 명령 가이드 등 전체 버튼 완전 매트릭스.
- 각 버튼은 화면 전환, 시트 열림, 토스트 중 하나를 만들어야 한다.
- 콘솔 에러 0을 유지해야 한다.
