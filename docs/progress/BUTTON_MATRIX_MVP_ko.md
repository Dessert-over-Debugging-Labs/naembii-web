# MVP 버튼 클릭 검증 매트릭스

작성일: 2026-07-01

## 범위

이번 라운드는 최신 변경과 직접 연결된 핵심 버튼을 CDP로 실제 클릭 검증했다. 전체 버튼 완전 매트릭스는 `MVP-06` Ralph 스토리로 남긴다.

## 검증 결과

### 2026-07-02 재검증

CDP로 화면별 표시 버튼을 실제 클릭했다. 각 버튼은 화면 전환, 시트 열림, 토스트, 상태 변경 중 하나를 만들었고 콘솔 에러는 0이었다.

| 묶음 | 범위 | 클릭 수 | 실패 |
| --- | --- | ---: | ---: |
| A | 홈, 요리북, 등록 시트 | 10 | 0 |
| B | 상세, cook3, 타이머, 재료, 핸즈프리 | 13 | 0 |
| C | 완료, 팁 남기기, 후기 작성 | 9 | 0 |

스크린샷:
- `scripts/ralph/screenshots/mvp06-button-matrix-a.png`
- `scripts/ralph/screenshots/mvp06-button-matrix-b.png`
- `scripts/ralph/screenshots/mvp06-button-matrix-c.png`

검증한 주요 액션:
- 홈: 영상 링크 등록, 검색 안내, 카테고리, 전체, 하단 등록, 하단 요리북.
- 요리북: 필터, 최근 조리 카드.
- 등록 시트: 취소, 입력 완료.
- 상세: 공유, 저장, 영상 링크 상태, 탭, 후기 시트, 요리 시작.
- cook3: 영상 링크 상태, 색상칩, 타이머, 타이머 시작, 재료, 핸즈프리, 요리 완성.
- 완료/팁/후기: 팁 화면, 후기 시트, 공유 안내, 홈 이동, 팁 태그, 팁 등록, 나중에 하기, 별점, 후기 등록 validation.

## 남은 검증/잔여 리스크

- 숨김 상태의 오버레이 닫기 버튼, 파일 업로드 input처럼 자동화가 직접 값을 넣기 어려운 요소는 정적 존재/aria 범위로만 본다.
- 새 화면이나 새 버튼을 추가할 때는 이 문서에 버튼명을 추가해 재검증한다.
