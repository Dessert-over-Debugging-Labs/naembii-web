### [F0] 연결 스파이크 — home end-to-end 파이프라인 증명 · 2026-07-02

- **무엇을**: 연결→그리기→read-back→export 파이프라인이 실제로 동작함을 홈 1화면(최소 구성)으로 증명.
- **어떻게(주요 노드)**: `cook/home` 프레임 `nodeId=2004:2`(390×844, 자식 9개).
  - 상태바 시간(2004:3), 오렌지 로고 마크(2004:4), `Cookflow` 워드마크(2004:5)·서브(2004:6),
    헤드라인 `오늘 뭐 만들까요?`(2004:7), 오렌지 CTA 카드(2004:8)+아이콘(2004:10)+제목/서브(2004:9/2004:11).
  - 좌표 함정 해결: `create_*`(parentId)는 부모-상대 좌표라 자식이 프레임 밖으로 밀림 → `move_node`(절대 좌표)로 재배치.
- **확인(export)**: `scripts/ralph-figma/exports/home_spike.png`(780×1688 유효 PNG). export 툴은 base64 인라인 반환이라
  세션 transcript JSONL에서 IHDR 치수(780×1688)로 골라 저장. 소스 `source-screens/home.png`와 영역 대응
  (상태바 → 로고 헤더 → 헤드라인 → 오렌지 CTA) 확인.
- **게이트**: G0 ✅ / G1 ✅ / G2 ✅(자식 9, 헤드라인 텍스트 노드) / G3 ✅ / G4 ✅(영어 누수 0·오렌지/크림 토큰·거짓 상태 없음).
- **다음**: F1-01 home 전체 이식(검색바·카테고리 원형칩 5개·추천 카드 2개·하단탭바까지 확장).
